# VERSIONING

Contract Lane v1.0.0 freeze policy:

1. No breaking change is allowed to any `*-v1` schema or hashing/signing semantics.
2. Incompatible changes require a new version suffix (for example `*-v2`).
3. Conformance is the compatibility gate.
4. Documentation must reflect actual conformance-enforced behavior and identifiers.

Frozen v1 surfaces:

- `agent-id-v1`
- `sig-v1`
- `evidence-v1`
- `amount-v1`
- `delegation-v1`
- `delegation-revocation-v1`
- `rules-v1`
- `proof-bundle-v1`
